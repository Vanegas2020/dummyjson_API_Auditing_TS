# dummyjson_API_Auditing_TS

## Overview

This is an API Testing Automation project generated by **Automation Test Generator**.
It uses **Playwright** as the core framework and **TypeScript** for scripting.

### Capabilities
- **Framework**: Playwright
- **Language**: TypeScript
- **Package Manager**: npm
- **Auth Strategy**: none
- **Test Level**: auditing
- **Reporting**: html, list

## Prerequisites

- Node.js 24.x
- npm

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure Environment:
   Rename `.env.example` to `.env` and fill in your secrets.
   ```bash
   cp .env.example .env
   ```

## Running Tests

### Run all tests
```bash
npm test
```

### Run specific test file
```bash
npx playwright test tests/01-public/<spec-file>.spec.ts
```

### Show Report
```bash
npm run test:report
```

## Project Structure

- `tests/`: Contains generated test specifications organized by phase.
- `tests/fixtures/`: Playwright fixtures for Auth and Context.
- `tests/helpers/`: Utilities for validation and schemas.
- `data/test-data.json`: Sample payloads for all endpoints.
- `.env`: Environment variables configuration.
- `playwright.config.ts`: Main configuration file.
